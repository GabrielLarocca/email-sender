# Email Sender Api

A utility API, written in node.js, for the purpose of sending emails.

## :wrench: Setup

- In `/api` folder, run `npm install` to install the dependencies;
- Need to set API keys in config.js file;
- To init the server, run `node index.js`;

## 👨‍💻 Technologies used

- Node.js;
- Express.js;
- Mongoose 6.1.7;
